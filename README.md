# My Theme Script :
A TUI script that helps you install Themes according to your Linux Distribution
<br><br>
We Have 2 Distributions currently :
*   Ubuntu
*   Linux Mint

<pre>
  $ git clone https://github.com/MHJedli/theme_script.git
  $ cd theme_script/
  $ bash setup.sh or ./setup.sh
</pre>
