# My Theme Script :
A TUI script that helps you install Themes according to your Linux Distribution
<br><br>
Current Script Directory Structure and the Available Themes in a tree representation :
```
.
└── src
    ├── Mint
    │   └── 21.3
    │       ├── Cinnamon-BigSur
    │       ├── Cinnamon-Ventura
    │       └── Windows-7
    └── Ubuntu
        └── 22.04
            └── GNOME-42
                ├── GTK-Graphite
                ├── My-Theme
                ├── MacOS V1
                ├── Windows-11
                └── Windows-Everforest-Dark

```
